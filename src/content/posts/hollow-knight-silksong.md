## 前言

自2019年Team Cherry公布**空洞骑士：丝之歌**已时隔6年之久，而就在这个月4号（9月4日），**丝之歌**也是正式上架了Steam，笔者作为一个**空洞骑士**老玩家那自然也是二话不说直接就buy了，结合最近一段时间的游玩体验再加上笔者个人对游戏的客观理解，今天来说说这个“从DLC演变成正统续作”的**丝之歌**到底适不适合入手。

> [!NOTE] 以下所有内容不会涉及游戏剧透，请放心阅读

笔者目前游戏大约50个小时，只玩了猎手和漫游者两个职能，先贴个小黑盒存档解析，目前在做全收集。

ps：但这真的不是笔者半个月没更新博客的原因\_(:з」∠)\_

## 关于游戏

> [!NOTE] 以下内容为个人观点，仅供参考，每个人对游戏理解都不一样，没有对错之分

**丝之歌**延续了前作**空洞骑士**的大部分要素，除了该死的游戏难度，TC你是觉得大家都是硬核高玩是吗！因此很多地方你能看到前作的影子，以下从不同方面来说说笔者对本作的一些游戏感受。

### 地图

游戏地图还是一如既往的大，笔者感觉比前作大不少但这同样也增加了跑尸的时间\_(:з」∠)\_

### 系统

本作取消了前作的**护符系统**，取而代之的是新的**纹章系统**（可以理解为职业切换）和**陷阱工具**系统（可以携带各种小道具辅助探索打怪），网上看到有玩家对这一变化表示不如前作的护符可进行多样化搭配，但笔者认为本作更强调的是合理搭配不同职业（纹章）和道具进行克敌制胜，只能说玩法各有各的优点，观点各有各的见解。

### 剧情

依旧是半碎片化叙事，虽说是正统续作，笔者个人认为不及前作的剧情，但不乏一些令人震撼的场景，可能称其为“剧情补充”会更合理？纯属个人观点，如果单看**丝之歌**本身的话，整体剧情还是不错的。

### 音乐与美术

绝赞！TC在这方面从不会令人失望！

### 本土化翻译

部分中文翻译不佳，笔者一个中国人居然有看不懂中文的一天，但TC承诺后续补丁改进，所以可以观望一下。

### 操作

仁者见仁智者见智，不同纹章的操作略有不同，入手的小伙伴自行体会。

> [!TIP] 强烈推荐使用手柄进行游玩！

### 难度

这笔者就不得不吐槽了！**本作比前作难，但难的不是boss！**而是包含但不限于以下内容：

-   伤害高到离谱的小怪 后期虫均打一下扣2滴血
    
-   大量专治手残的跳跳乐地图 玩了**丝之歌**，人人都能成为前作**苦痛之路**高手（bushi）
    
-   令人发指的长距离跑尸地图 ♪三百六十五里路呦~~~~~♪
    
-   各种收费椅子和陷阱椅子 TC你什么不学好！整这个是吧！
    
-   etc.
    

难度方面笔者建议先去玩前作，再来玩**丝之歌**，游戏体验也许会好一点。

### 价格

目前（截止2025年9月22日），游戏steam国区售价76元。

## 适不适合入手

> [!NOTE] 以下内容为个人观点，仅供参考，每个人对游戏理解都不一样，没有对错之分

### 适合入手

-   [x] 
    
    **平台跳跃**类型或**类银河恶魔城**类型游戏的高手
    
-   [x] 
    
    完美通关前作**空洞骑士**的老玩家（玩过前作真心建议入）
    
-   [x] 
    
    对自己的操作信心十足的小伙伴
    

### 视钱包情况入手

-   [x] 
    
    喜欢玩**平台跳跃**类型或**类银河恶魔城**类型的小伙伴
    
-   [x] 
    
    没玩过前作**空洞骑士**或只玩了一半没通关的小伙伴
    
-   [x] 
    
    对自己的操作没什么信心，但是想玩玩看的小伙伴
    

### 不适合入手

-   [x] 
    
    没接触过**平台跳跃**类型或**类银河恶魔城**类型游戏的小伙伴
    
-   [x] 
    
    没什么耐心，比较容易自暴自弃的小伙伴（真不建议浪费这个钱，为啥不去吃顿好的呢）
